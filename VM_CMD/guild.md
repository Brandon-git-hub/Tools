

這是在 `vmrun` 命令列工具中，用於控制虛擬機電源狀態的相關指令與參數整理：

| 電源指令 | 參數選項 | 說明 |
| :--- | :--- | :--- |
| **`start`** | `[ gui \| nogui ]` | 啟動虛擬機。預設為 `gui`，會以互動方式啟動並顯示虛擬機視窗；若使用 `nogui` 則不會顯示介面，適合用於背景執行或自動化腳本。**注意：** 啟動加密的虛擬機時必須使用 `nogui` 參數。 |
| **`stop`** | `[ hard \| soft ]` | 停止（關閉）虛擬機。`soft` 會向客體作業系統發出訊號並執行正常的關機指令碼（軟關機）；`hard` 則如同直接按下實體電源按鈕，強行關閉虛擬機電源（硬關機）。 |
| **`reset`** | `[ hard \| soft ]` | 重新啟動虛擬機。`soft` 會先執行關機指令碼後再正常重新啟動；`hard` 則強行復位並重新啟動虛擬機，不等待系統正常關閉。 |
| **`suspend`** | `[ hard \| soft ]` | 將虛擬機掛起。`soft` 會在掛起前透過 VMware Tools 執行系統腳本（例如在 Windows 釋放 IP，在 Linux 暫停網路連線）；`hard` 則直接掛起虛擬機，不執行任何腳本。 |
| **`pause`** | (無額外參數) | 暫停虛擬機。虛擬機會暫時停止運作（畫面反灰），且不會發送或接收網路封包。 |
| **`unpause`** | (無額外參數) | 恢復運行。讓處於 `pause` (暫停) 狀態的虛擬機繼續正常運作。 |
